# Delta Growth – WordPress Practical Assessment

This repository contains the custom WordPress theme and documentation developed as part of a practical assessment for the WordPress Developer role.

## Project Overview
- Platform: WordPress
- Page Builder: Elementor (Free)
- E-commerce: WooCommerce + Paystack (Test Mode)
- Custom Features: Testimonials, Carousel Display
- Design Approach: Mobile-first, clean digital agency layout

## Repository Structure
- /wp-content/themes/delta-growth – Custom theme files
- /documentation – Project documentation

## Setup Notes
- Clone repository into WordPress `wp-content/themes` directory
- Activate theme from WordPress dashboard
- Required plugins listed in documentation
